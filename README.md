# ``Run project``
## Run Django server
  1) open "ELISA/DjangoServer/" directory
  2) run "source venv/bin/activate"
  3) run "cd ELISA_Django_Server"
  4) run "pip3 install -r requirements.txt"
  5) run "python3 manage.py runserver"
